Guide d'importation des articles

Bonjour {{ first_name }},

Vous trouverez ci-joint un fichier modèle pour l'importation des articles. Notre système accepte les formats CSV, Excel (.xls) et Excel (.xlsx). Voici les spécifications détaillées de chaque champ :

Formats supportés
  • CSV (.csv) - Fichier texte avec séparateur point-virgule (;)
  • Excel (.xls) - Format Excel 97-2003
  • Excel (.xlsx) - Format Excel moderne (2007+)

Structure du fichier
Réf Obligatoire. Référence unique de l'article. Elle est aussi utilisée comme code barre lors de l'import historique.
Désignation Obligatoire. Nom de l'article affiché dans les articles, le stock et la caisse.
Famille Optionnel. Code de la famille article. Si elle n'existe pas, elle sera créée automatiquement.
Unité Vente Optionnel. Unité de vente, par exemple Pièce, Carton, Kg ou Litre.
P Achat Obligatoire. Prix d'achat en format numérique. Utilisez le point ou la virgule pour les décimales.
P Gros Obligatoire. Prix de gros en format numérique.
P Details Obligatoire. Prix détail en format numérique.
P Comptoir Obligatoire. Prix comptoir utilisé dans la caisse.
Stock Alert Obligatoire. Stock minimum par défaut pour les notifications.
Stock Optionnel. Quantité initiale enregistrée dans le stock MBR pendant l'import.
Date expiration Optionnel. Date d'expiration. Formats acceptés : AAAA-MM-JJ, JJ/MM/AAAA ou JJ-MM-AAAA.
Durée Optionnel. Durée de conservation en jours.

Points importants
  • La première ligne doit contenir les en-têtes de colonnes
  • Pour les fichiers CSV, utilisez le point-virgule (;) comme séparateur
  • Pour les fichiers CSV, l'encodage doit être UTF-8
  • Pour Excel, utilisez la première feuille du classeur ou une feuille nommée "F"
  • Les nombres décimaux peuvent utiliser le point (.) ou la virgule (,)
  • Les colonnes Réf, Désignation, P Achat, P Gros, P Details, P Comptoir et Stock Alert doivent être renseignées
  • Les familles et unités absentes seront créées automatiquement pendant l'import
  • Vérifiez les références avant import pour éviter les doublons de codes barres
